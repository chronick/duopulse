# V3 Design Documentation

DuoPulse V3 ("Pulse Field V3") introduced the weighted pulse field concept with BROKEN and DRIFT controls.

## Primary Documentation

V3 design is documented in the task files:

- [Task 13: Pulse Field V3](../../../tasks/completed/13-pulse-field-v3.md) - Main V3 implementation
- [Task 14: V3 Ratchet Fixes](../../../tasks/completed/14-v3-ratchet-fixes.md) - Bug fixes and refinements

## Key Features

- **Weighted Pulse Field**: Probabilistic hit placement based on metric weights
- **BROKEN Control**: Timing displacement for syncopation
- **DRIFT Control**: Pattern evolution across phrases
- **COUPLE Interlock**: Voice relationship modes
