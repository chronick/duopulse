# Wrap Session
Wrap up the current stage of our task implmentation, commit, take notes.

## Steps
1. **Verify Tests and Build** Run `make test` and `make` to verify correctness.
2. **Verify Linting Errors** Ensure there are no linting errors. Confirm before fixing.
3. **Add Comments** Add comments to the current task in `docs/tasks` to explain changes done during this session.
4. **Commit Code** Commit code with git to this branch with a good commit message. Remain on this branch.
